## 보안 조치

1. **전송 암호화**: 모든 데이터는 HTTPS로 암호화 전송
2. **저장 암호화**: 비밀번호는 bcrypt 해시 저장
3. **접근 제어**: 역할 기반 접근 제어 (RBAC)
4. **세션 관리**: 자동 로그아웃, 세션 만료
