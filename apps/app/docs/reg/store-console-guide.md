## 앱 권한 설정

### 사용하는 권한

| 권한 | Android | iOS | 용도 |
|-----|---------|-----|-----|
| 인터넷 | INTERNET | 기본 허용 | 서버 통신 |
| 파일 다운로드 | WRITE_EXTERNAL_STORAGE* | 파일 앱 접근 | Excel 다운로드 |
| 파일 업로드 | READ_EXTERNAL_STORAGE* | 문서 선택기 | Excel 업로드 |

*Android 10+ (API 29+)는 Scoped Storage 사용으로 별도 권한 불필요

### 사용하지 않는 권한

| 권한 | 설명 |
|-----|-----|
| 카메라 | 사용 안 함 |
| 위치 | 사용 안 함 |
| 연락처 | 사용 안 함 |
| 마이크 | 사용 안 함 |
| 전화 | 사용 안 함 |
| SMS | 사용 안 함 |
| 캘린더 | 사용 안 함 |

### Android (AndroidManifest.xml)

```xml
<!-- 필수 권한 -->
<uses-permission android:name="android.permission.INTERNET" />

<!-- 파일 다운로드 (Android 9 이하) -->
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE"
    android:maxSdkVersion="28" />
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE"
    android:maxSdkVersion="32" />
```

### iOS (Info.plist)

```xml
<!-- 파일 접근 설명 (필요시) -->
<key>NSDocumentsFolderUsageDescription</key>
<string>Excel 파일 업로드/다운로드를 위해 파일 접근이 필요합니다.</string>
```
